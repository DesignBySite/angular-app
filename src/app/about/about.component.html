An Angular 5 goals app

This is a simple Angular 5 app created for practice. Following the guide from coursetro.com, I learned about the new advantages of Angular 5 and how it carries over many similarities from Angular 2.

I learned about any differences for Angular 5:

  • Components
  • Interpolation
  • Property, Event, and Two-Way Data Binding
  • New Animation Guidelines
  • Routing
